# coal-dashboard

![coal-dashboard](frontend/.github/screenshot1.png)

Dashboard для отслеживания и предсказывания самовозгораний угля при открытом его хранении.

## Структура проекта

`machine_learning` - исследовательская составляющая проекта

`frontend` - визуальная составляющая проекта

## Запуск фронтенда

```bash
git clone https://github.com/R3dKar/coal-dashboard.git
cd coal-dashboard/frontend
npm ci
npm run dev --open
```
